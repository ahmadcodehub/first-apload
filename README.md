Student Management System (Python)
Overview

This is a simple Student Management System built using Python.
It helps to manage student records by adding, viewing, searching, and deleting students.
All data is stored in a text file (students.txt), so the data is saved even after closing the program.

Features
Add new students
View all students
Search student by name
Delete student by name
Data is saved using file handling
Menu-based system for easy use
Technologies Used
Python
File Handling
Object-Oriented Programming (OOP)
How to Run

Run the Python file:

python student management system.py
